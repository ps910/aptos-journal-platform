# Wallet Setup Guide

## Overview
The Aptos Journal Platform requires a Web3 wallet to interact with the blockchain. This guide will help you set up your wallet.

## Supported Wallets

We recommend **Petra Wallet** for the best experience with Aptos:

### 1. **Petra Wallet** (Recommended)
- **Website**: https://petra.app/
- **Download**: Available for Chrome, Firefox, Edge, and Brave
- **Network**: Aptos Testnet
- **Features**: Easy to use, good UI, wide adoption

### 2. **OKX Wallet**
- **Website**: https://www.okx.com/web3
- **Download**: Available for Chrome and Firefox
- **Network**: Aptos Testnet

### 3. **Martian Wallet**
- **Website**: https://martianwallet.xyz/
- **Download**: Available for Chrome

## Step-by-Step Setup

### Step 1: Install Wallet Extension
1. Go to https://petra.app/
2. Click "Download" or install from your browser's extension store
3. Complete the installation

### Step 2: Create a New Account
1. Open the Petra Wallet extension
2. Click "Create New Wallet"
3. Save your seed phrase securely (write it down!)
4. Create a strong password

### Step 3: Get Testnet APT
1. Go to Aptos Faucet: https://faucet.testnet.aptoslabs.com/
2. Paste your wallet address
3. Request APT tokens (you'll get 1 APT every 24 hours)
4. Tokens should arrive in a few seconds

### Step 4: Connect Wallet to Aptos Journal
1. Go to http://localhost:3000 (or your deployment URL)
2. Log in to your account
3. Click "Connect Wallet" in the navbar
4. Approve the connection in your wallet extension
5. Done! Your wallet is now connected

## Troubleshooting

### "Wallet Extension Not Found"
- Make sure you've installed Petra Wallet or another supported wallet
- Refresh the page after installing
- Try a different browser if the extension isn't appearing

### "Connection Failed"
- Make sure your wallet is unlocked
- Try disconnecting and reconnecting
- Clear your browser cache and try again
- Restart the browser extension

### "Insufficient Balance"
- You need at least 0.1 APT to create a journal entry
- Get free testnet APT from: https://faucet.testnet.aptoslabs.com/

### Wallet Not Showing Balance
- Refresh the page
- Make sure you're connected to Aptos Testnet
- Check that you have testnet APT tokens
